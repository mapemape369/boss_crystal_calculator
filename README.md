# boss_crystal_calculator
boss_crystal_calculator

처음 실행시 좌측 상단에 '추가' 클릭, 프리셋 이름은 편한대로 하면 됩니다
주보돌이도 같은 방식으로 추가 가능
사용시 데이터 저장용 파일 'presets_save.json'가 생성됨 삭제 금지

수익 요약 설명

현재 주간 수익 > 현재 프리셋 캐릭터의 주간 수익 (검마x)


현재 월간 수익 > 현재 프리셋 캐릭터의 주간 수익 (검마0)


전체 주간 수익 > 모든 프리셋 통합 주간 수익 (검마x)


전체 월간 수익 > 모든 프리셋 통합 월간 수익 (검마o)


//////////////////


주간 0/14 > 프리셋 당 보스 최대 14개 체크 가능


총 0/180 > 검은 마법사는 주간 최대 14개에는 포함되지 않으나, 180개에는 포함됨


수익 > 주간 + 검은 마법사의 결정 가격


이름 변경 > 원하는 프리셋에서 이름 변경을 누르고 수정하면 프리셋 이름을 바꿀 수 있음


삭제 > 삭제하고 싶은 프리셋 상태에서 누르면 해당 프리셋 삭제


+ 프리셋은 최대 20개 까지 생성 가능하게 설정했으나, 안해봐서 제대로 작동하는지는 모름
